API Setup

Getting Your OpenRouter API Key

Why OpenRouter? OpenRouter provides access to multiple AI models (Claude, GPT-4, Gemini, etc.) through a single API, giving you flexibility and choice.

  1. 1.
    Visit openrouter.ai and create a free account
  2. 2.
    Navigate to the API Keys section in your dashboard
  3. 3.
    Click "Create Key" and give it a descriptive name
  4. 4.
    Copy your API key (it starts with "sk-or-")

Configuring the App

  1. 1.
    Open the AI Prompt Generator
  2. 2.
    Scroll to the "API Settings" section
  3. 3.
    Paste your API key in the "OpenRouter API Key" field
  4. 4.
    Select your preferred AI model from the dropdown
  5. 5.
    Click "Save" to store your settings

Security Note: Your API key is stored locally in your browser's cookies. It's never sent to any server except OpenRouter when making API calls.

Creating Basic Prompts

The app uses a structured approach to prompt creation with three key components: Role, Task, and Format.

ROLE

Defines who the AI should act as

Examples: Marketer, Teacher, Analyst, Writer

TASK

Specifies what the AI should do

Examples: Write Article, Create Summary, Generate Ideas

FORMAT

Determines the output structure

Examples: Table, List, JSON, Essay

Step-by-Step Example

Let's create a prompt for generating marketing content:

  1. 1.
    Select Role: "Marketer"
  2. 2.
    Choose Task: "Social Media Post"
  3. 3.
    Pick Format: "A List"
  4. 4.
    Add Seed Prompt: "Promote a new eco-friendly water bottle for fitness enthusiasts"

Generated Base Prompt:

"Acting as a Marketer, perform Social Media Post in A List. Seed prompt: Promote a new eco-friendly water bottle for fitness enthusiasts"

Pro Tip: Start simple with basic selections, then use the AI enhancement feature to make your prompts more sophisticated and effective.

AI Enhancement

Transform your basic prompts into professional-grade instructions using AI enhancement.

Before Enhancement

"Acting as a Marketer, perform Social Media Post in A List. Seed prompt: Promote a new eco-friendly water bottle for fitness enthusiasts"

After Enhancement

"As an experienced digital marketer specializing in sustainable products, create a comprehensive social media campaign for an eco-friendly water bottle targeting fitness enthusiasts. Provide your response as a structured list that includes: compelling headlines, key benefits, target audience insights, hashtag recommendations, and call-to-action suggestions."

How Enhancement Works

  1. 1.
    Your basic prompt is sent to the selected AI model
  2. 2.
    The AI analyzes the structure and intent
  3. 3.
    It adds specificity, context, and clarity
  4. 4.
    Returns an improved version that maintains your original intent

Enhancement Features:

  • • Adds specific instructions and constraints
  • • Improves clarity and reduces ambiguity
  • • Includes relevant context and examples
  • • Optimizes for better AI response quality

Troubleshooting

API Key Issues

"Please enter your OpenRouter API key" error appears

Solution: Ensure your API key is correctly entered and saved. Check that it starts with "sk-or-" and hasn't expired.

Enhancement Not Working

The "Enhance with AI" button doesn't respond

Solution: Check your internet connection, verify API key validity, and ensure at least one field is filled before enhancing.

Validation Errors

Validation results show parsing errors

Solution: Try a different model (some handle JSON output better) or reduce the complexity of your validation criteria.

Storage Issues

Favorites or history not saving

Solution: Check if local storage is enabled in your browser. Private/incognito mode may prevent data persistence.

Still Need Help?

Check the browser console (F12) for detailed error messages, or try refreshing the page and re-entering your settings.
